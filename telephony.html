Облачная телефония

Для клиентов, у которых динамично развивающийся бизнес мы рекомендуем использовать Облачную телефонию. Данное решение дает возможность получить все достоинства IP-телефонии без первоначальных затрат на дорогостоящее оборудование.

Наши сервера находятся в дата-центрах, что позволяет давать максимальный аптайм сервиса - до 99.9%. Репликация серверов, бесперебойное питание, несколько каналов интернет - обеспечивает высокий уровень отказоустойчивости.

Для подключения GSM-каналов, мы предлагаем в аренду слоты для SIM-карточек.

Виртуальная АТС
Коробочная версия

Для тех, кто хочет использовать собственное оборудование у нас есть предложение - коробочная версия виртуальной IP-АТС.

Мы устанавливаем оборудование “под ключ”, а также готовы сопровождать Вашу телефонию, вождать Вашу телефонию, предлагая обслуживание, обновление системы.

В коробочную версию входит:

Сервер - мы подберем для Вас оборудование соответствующее требованиям по стоимости, надежности, отказоустойчивости

Настройка программной части, веб-интерфейс с личным кабинетом, статистикой звонков и записью телефонных разговоров.

GSM-шлюз - оборудование, для подключения мобильных SIM-карт в IP-АТС

IP-телефон - телефонный аппарат, для работы с IP-АТС, условно разделяем на 5 типов:

  • Начального уровня, базовый аппарат
  • Многофункциональный, корпоративного уровня
  • Премиум линейка, с сенсорным экраном, ОС Android
  • DECT-телефон - переносной, беспроводной IP-телефон
  • Softphone - Программный IP + гарнитура - решение для колл-центров, когда нужны свободные руки для операторов
  • Аналоговый телефон + FXS порт

Аппаратные IP-АТС

Также мы имеем богатый опыт интеграции аппаратных IP-АТС. Есть авторизация Panasonic, что позволяет давать расширенную гарантию на АТС Panasonic.

Аппаратные IP-АТС - решения для компаний, которым нужна телефония без сложных алгоритмов, надежные и шаблонные. Единая архитектура, программное обеспечение, разветвленная сеть сервисных центров. Аппаратные IP-АТС состоят из аппаратной части - Базовый блок, платы расширения и программная часть, которая лицензируется.
